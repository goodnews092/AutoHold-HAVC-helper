# 아이오닉5(22my) 스마트오토홀드/공조편의

1. 지능형 오토홀드(AutoHold) 자동 제어
상황 맞춤형 제어 로직: 차량의 현재 속도, 기어 상태, 브레이크 조작 여부를 실시간으로 모니터링하여
오토홀드를 자동으로 활성화하거나 해제합니다.
주차 및 주행 안전 방어 로직: 후진(R단) 체결 시 오토홀드를 자동으로 해제하여 주차 편의성을 높였습니다.
또한 크루즈 컨트롤 및 제한 속도 기능 활성화 상태(CAN ID 0x2E1 등)를 식별하여 순정 스티어링휠 버튼(CANCEL)의
주행 중 발생할 수 있는 오작동을 차단합니다.
- 주차 중 P단이 아닌 상태에서 시동을 종료하면 다음 시동시 오토홀드는 작동하지 않습니다.
- 차량순정의 오토홀드 버튼을 눌러 재활성 할 수 있습니다.

1. 차량 공조(HVAC) 및 편의 장치 제어
시트 및 핸들 통합 관리: 운전석과 조수석의 열선/통풍 시트(1~3단계) 및 2열 열선, 핸들 열선을 디스플레이에서 원격으로 제어합니다.
공조 시스템 상태 동기화: A/C, HEAT, SYNC, DRIVER ONLY 등의 공조 상태를 모니터링하고 제어 명령을 양방향으로 동기화합니다.
인터리빙 인젝션(Interleaved Burst): 차량 CAN 버스에 제어 신호를 보낼 때, 타겟 제어 패킷과 비프음 패킷을 교차 전송(Interleaving)하는 알고리즘을 적용하여 통신 딜레이를 최소화하고 조작 응답성을 극대화했습니다.

1. 무선 터치 디스플레이 UI (LVGL & ESPHome)
ESP-NOW 실시간 통신: Wi-Fi 라우터 없이 메인 모듈과 디스플레이 모듈 간 ESP-NOW 프로토콜을 사용해 지연 없는 실시간 무선 통신을 수행합니다.
차량 환경 연동 디스플레이: * 오토라이트 연동(Dimming): 차량의 라이트 상태(CAN ID 0x413)를 수신하여 주/야간에 맞춰 디스플레이 밝기를 자동으로 조절합니다.
스마트 스크린세이버: 차량 내부의 시계/날짜 정보(CAN ID 0x4F0)를 실시간으로 파싱하여 스크린세이버 화면에 동기화합니다.
사용자 편의 UI 로직: 시트 열선/통풍 버튼을 길게 누르는(Long Press, 0.75초) 동작만으로 즉시 전원이 차단(OFF)되도록 직관적인 조작성을 부여했습니다.

1. 시스템 안정성 및 최적화 (Failsafe & Hardware)
간편 페어링 및 설정 유지: 핸들 버튼의 롱프레스 또는 기기의 물리 버튼을 통해 간편하게 하드웨어 페어링 모드로 진입할 수 있으며, 페어링된 MAC 주소 및 Wi-Fi 채널, 비프음 설정 내역은 NVS(비휘발성 메모리)에 안전하게 저장됩니다.
오류 자동 복구: Watchdog Timer(WDT) 및 CAN 버스 에러 자동 복구(Recovery) 로직을 적용하여 실차 주행 환경에서 발생할 수 있는 일시적인 통신 단절을 방지합니다.
하드웨어 공간 효율 극대화: 부피를 차지하는 기존 12V 릴레이를 배제하고 PC817 포토커플러를 설계에 적용함으로써 콤팩트하면서도 전기적으로 분리된 안정적인 제어 환경을 구축했습니다.
